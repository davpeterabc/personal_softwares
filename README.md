# Windows客户端配置步骤

## 1.解压文件

<img src="https://github.com/jqhph/dcat-admin/workflows/Laravel%20Dusk/badge.svg" alt="Build Status">


## 2.配置客户端程序

双击运行客户端

点击图标，打开软件界面-添加服务器
